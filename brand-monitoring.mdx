## Setting Up Brand Monitoring

Configure your brand monitoring to track relevant keywords and platforms.

<Steps>
  <Step title="Define Keywords" icon="search">
    Specify keywords, hashtags, and phrases related to your brand.
  </Step>
  <Step title="Select Platforms" icon="settings">
    Choose social media and web platforms to monitor.
  </Step>
  <Step title="Set Alerts" icon="bell">
    Configure alerts for important mentions or sentiment changes.
  </Step>
  <Step title="Review Dashboard" icon="eye">
    Access your personalized brand monitoring dashboard.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
const calify = require('calify-sdk');
const client = new calify.Client({ apiKey: 'your-api-key' });

const monitorConfig = {
  keywords: ['brand name', '#hashtag'],
  platforms: ['twitter', 'linkedin'],
  alertThreshold: 'high'
};

await client.createBrandMonitor(monitorConfig);
```
  ```python
from calify_sdk import CalifyClient

client = CalifyClient(api_key='your-api-key')

monitor_config = {
    'keywords': ['brand name', '#hashtag'],
    'platforms': ['twitter', 'linkedin'],
    'alert_threshold': 'high'
}

client.create_brand_monitor(monitor_config)
```
</CodeGroup>

## Reports and Exports

Generate comprehensive reports on your brand performance.

<CodeGroup tabs="cURL,Java">
  ```bash
curl -X GET "https://api.calify.com/brand/reports?period=monthly" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -o brand_report.pdf
```
  ```java
import com.calify.sdk.CalifyClient;
import com.calify.sdk.BrandReport;

CalifyClient client = new CalifyClient("your-api-key");
BrandReport report = client.getBrandReport("monthly");
report.exportToFile("brand_report.pdf");
```
</CodeGroup>

## Best Practices

Maximize the value of brand monitoring with these strategies.

- Monitor relevant industry keywords beyond just your brand name
- Set up alerts for negative sentiment to respond quickly
- Use insights to refine your content strategy
- Compare performance across different platforms

<Callout kind="info">
  Brand monitoring data is updated in real-time for accurate insights.
</Callout>

## Troubleshooting

Common issues and solutions for brand monitoring.

| Issue | Solution |
|-------|----------|
| Missing mentions | Expand keyword list and platform coverage |
| Inaccurate sentiment | Review and adjust sentiment analysis settings |
| Alert overload | Fine-tune alert thresholds and keywords |
