## Adding Connections

Calify makes it easy to add new connections through multiple methods.

<Steps>
  <Step title="Manual Entry" icon="plus">
    Add connections manually by entering their contact information.
  </Step>
  <Step title="Import from Email" icon="mail">
    Import contacts from Gmail, Outlook, or other email providers.
  </Step>
  <Step title="Social Media Sync" icon="share">
    Sync connections from LinkedIn or other social platforms.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
const calify = require('calify-sdk');
const client = new calify.Client({ apiKey: 'your-api-key' });

const newConnection = {
  name: 'John Doe',
  email: 'john@example.com',
  type: 'professional',
  notes: 'Met at conference'
};

await client.createConnection(newConnection);
```
  ```python
from calify_sdk import CalifyClient

client = CalifyClient(api_key='your-api-key')

new_connection = {
    'name': 'John Doe',
    'email': 'john@example.com',
    'type': 'professional',
    'notes': 'Met at conference'
}

client.create_connection(new_connection)
```
</CodeGroup>

## Best Practices

Maximize the value of your Calify connections with these tips.

- Regularly update connection information
- Schedule periodic check-ins with important contacts
- Use notes to remember personal details
- Leverage tags for quick filtering and segmentation

<CodeGroup tabs="cURL,Go">
  ```bash
curl -X GET "https://api.calify.com/connections?type=professional" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
  ```go
package main

import (
	"fmt"
	"github.com/calify/calify-go"
)

func main() {
	client := calify.NewClient("your-api-key")
	connections, err := client.GetConnections(calify.ConnectionFilter{Type: "professional"})
	if err != nil {
		panic(err)
	}
	fmt.Printf("Found %d professional connections\n", len(connections))
}
```
</CodeGroup>
