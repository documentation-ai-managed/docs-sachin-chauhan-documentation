## Creating Scheduled Interactions

Plan your networking activities with Calify's intuitive scheduling interface.

<Steps>
  <Step title="Select Connection" icon="user">
    Choose the connection you want to interact with.
  </Step>
  <Step title="Choose Interaction Type" icon="message-square">
    Select from call, meeting, email, or custom interaction types.
  </Step>
  <Step title="Set Date and Time" icon="clock">
    Pick a convenient date and time for the interaction.
  </Step>
  <Step title="Add Details" icon="edit">
    Include notes, agenda items, or preparation reminders.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Ruby">
  ```javascript
const calify = require('calify-sdk');
const client = new calify.Client({ apiKey: 'your-api-key' });

const schedule = {
  connectionId: 'conn_123',
  type: 'meeting',
  scheduledAt: '2024-10-20T14:00:00Z',
  notes: 'Discuss partnership opportunities'
};

await client.createSchedule(schedule);
```
  ```ruby
require 'calify_sdk'

client = Calify::Client.new(api_key: ENV['CALIFY_API_KEY'])

schedule = {
  connection_id: 'conn_123',
  type: 'meeting',
  scheduled_at: '2024-10-20T14:00:00Z',
  notes: 'Discuss partnership opportunities'
}

client.create_schedule(schedule)
```
</CodeGroup>

## Managing Schedules

Update, reschedule, or cancel interactions as needed.

<CodeGroup tabs="Python,PHP">
  ```python
from calify_sdk import CalifyClient

client = CalifyClient(api_key='your-api-key')

# Update a schedule
updated_schedule = {
    'id': 'sched_456',
    'scheduled_at': '2024-10-21T10:00:00Z',
    'notes': 'Updated meeting time'
}
client.update_schedule(updated_schedule)

# Cancel a schedule
client.cancel_schedule('sched_456')
```
  ```php
<?php
require 'vendor/autoload.php';

use Calify\CalifyClient;

$client = new CalifyClient('your-api-key');

// Update a schedule
$updatedSchedule = [
    'id' => 'sched_456',
    'scheduled_at' => '2024-10-21T10:00:00Z',
    'notes' => 'Updated meeting time'
];
$client->updateSchedule($updatedSchedule);

// Cancel a schedule
$client->cancelSchedule('sched_456');
?>
```
</CodeGroup>

## Best Practices

Optimize your scheduling workflow with these recommendations.

- Schedule regular check-ins with key connections
- Use different interaction types for variety
- Include preparation notes for each schedule
- Review and update schedules weekly
